To run the annotation generator notebook, follow these steps:

1. Make sure you have the necessary dependencies installed. You can find the list of dependencies in the `requirements.txt` file.

2. Open the notebook in your preferred Jupyter environment.

3. Set the desired parameters for the annotation generation, such as input data paths, output directory, and annotation settings.

4. Run the notebook cells sequentially to generate the annotations.

5. Once the notebook has finished running, you will find the generated annotations in the specified output directory.

Regarding the folder structure, please refer to the `dir.txt` file for the desired directory structure. It will provide you with the information on how to organize your files and folders within the project.
